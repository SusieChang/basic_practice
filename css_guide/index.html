CSS权威指南

元素

置换元素和非置换元素

置换元素指用来置换元素内容的部分不由文档内容直接表示，而是根据属性决定。比如img、input、textarea等

HTML元素大部分是非置换元素，即元素的内容由文档内容直接表示。

行级元素和块级元素

行内元素的宽度随内容改变，相邻的行内元素排列在同一行，直到一行容纳不下。无法设置宽高。margin和padding设置只在左右方向上有作用。常见的行内元素有：

  • b, big, i, small, tt
  • abbr, acronym, cite, code, dfn, em, kbd, strong, samp, var
  • a, bdo, br, img, map, object, q, script, span, sub, sup
  • button, input, label, select, textarea

块级元素占据其父元素（容器）的整个空间。默认情况下，块级元素会新起一行。一般块级元素可以包含行内元素和其他块级元素。一些块级元素：

  • blockquote（引用）、address（联系方式信息）、article（文章内容）、audio（音频）、video（视频）
  • aside、footer、canvas、header、section
  • div、p、ul、ol、h1、h2、...
