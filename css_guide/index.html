元素

置换元素和非置换元素

置换元素指用来置换元素内容的部分不由文档内容直接表示，而是根据属性决定。比如img、input、textarea等

HTML元素大部分是非置换元素，即元素的内容由文档内容直接表示。

行级元素和块级元素

行内元素的宽度随内容改变，相邻的行内元素排列在同一行，直到一行容纳不下。无法设置宽高。margin和padding设置只在左右方向上有作用。常见的行内元素有：

  • b, big, i, small, tt
  • abbr, acronym, cite, code, dfn, em, kbd, strong, samp, var
  • a, bdo, br, img, map, object, q, script, span, sub, sup
  • button, input, label, select, textarea

块级元素占据其父元素（容器）的整个空间。默认情况下，块级元素会新起一行。一般块级元素可以包含行内元素和其他块级元素。一些块级元素：

  • blockquote（引用）、address（联系方式信息）、article（文章内容）、audio（音频）、video（视频）
  • aside、footer、canvas、header、section
  • div、p、ul、ol、h1、h2、...

CSS属性display

取值：display: [<display-outside> | <display-inside> | <display-listitem> | <display-internal> | <display-box> | <display-legacy>]

  • <display-outside> 指定了元素的外部显示类型，实际上就是其在流式布局中的角色。block | inline | run-in
  • <display-inside> 指定了元素的内部显示类型，它们定义了元素内部内容的格式化上下文的类型。flow | flow-root | table | flex | grid | ruby
  • <display-listitem> 将这个元素的外部显示类型变为 block 盒，并将内部显示类型变为多个 list-item inline 盒。<display-outside> ? && [ flow | flow-root ]? && list-item
  • <display-internal> 像 table 和 ruby 这样的布局模型有着复杂的内部结构，因此它们的孩子和后面的元素可能具有多个角色。这一类关键字就是用来定义这些“内部”显示类型。table-row-group | table-header-group | table-footer-group | table-row | table-cell | table-column-group | table-column | table-caption | ruby-base | ruby-text | ruby-base-container | ruby-text-container
  • <display-box> 这些值定义元素是否完全生成显示盒。contents | none
  • <display-legacy> CSS 2 对于 display 属性使用单关键字语法。inline-block | inline-list-item | inline-table | inline-flex | inline-grid

把CSS应用到HTML上

link标签

  • rel：关系，比如stylesheet、alternate stylesheet
  • type：样式表格式，最好指明，比如 text/css
  • media：指明媒体的类型
  • title：默认Default，会根据选择使用样式表

style元素

  • 指明type属性
  • @import在顶部引入外部样式表

属性选择符匹配子串

  • [ foo|='bar' ] 选择的元素有foo属性，且其值以bar和一个英文破折号开头，或者值就是bar本身
  • [ foo~='bar' ] 选择的元素有foo属性，且其值是包含bar这个词的一组词
  • [ foo*='bar' ] 选择的元素有foo属性，且其值包含子串bar
  • [ foo^='bar' ] 选择的元素有foo属性，且其值以bar开头
  • [ foo$='bar' ] 选择的元素有foo属性，且其值以bar结尾
